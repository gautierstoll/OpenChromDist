CHROMOSOME=

CHRLENGTH=

frag\+File is a list a ATAC hits.

bar\+Code\+File contains a list of barcodes separated by newline.

wind\+Eval is the window size (in bp) on which probability distribution is evaluated given a peak. It is an estimation parameters that should be mixed with WINDSIZE which defines the sd of the distribution.
